(static) search(terms, options) → {Array}

Source:
  • fluro.content.js, line 50
Runs a search from the Fluro server and returns the results
Example
FluroContent.search('Wonder', {limit:5, types:['song', 'album', 'tag']}).then(function(results) {
 //Will return a nested array with up to 5 results for each type
 //[{_type:'Song', results:[{title:"Wonder"...}]}, {_type:'Album', results:[{title:"Wonder"...}]}]
})

FluroContent.search('Wonder', {limit:5}).then(function(results) {
 //Will return an array of up to 5 items the user has access to view that match the search terms
 //[{title:"Wonder", _type:'article', definition:'song'...}, {title:"Wonder", _type:'article', definition:'album'...}]
})
Parameters:
Name Type Description
terms String The keywords to search for
options Object Extra Configuration and options for how to search the database and how to render the results
Properties
Name Type Description
limit Object How many results should be returned. Defaults to 10
types Array Specify types or definition names for which items should be searched for
showQuery Boolean If true will return the query used to search instead of the search results themselves
Returns:
An array of content items that match the search, if options.types is specified will be a nested array of results for each type
Type
Array
